# محاسب مشروعي

تطبيق ويب احترافي لإدارة حسابات المشاريع — مع ربط Easy Order وإرسال التقارير للعملاء عبر واتساب.

## المميزات

### الحسابات الشاملة
- **لوحة التحكم**: إجمالي الإيرادات والمصروفات والرصيد مع آخر الحركات.
- **الإيرادات والمصروفات**: تسجيل مع فئات (مبيعات، موردين، رواتب، إيجار، …).
- **الفواتير**: إصدار فواتير، تتبع الدفع، وإرسال تفاصيل الفاتورة للعميل عبر واتساب.
- **العملاء**: قائمة عملاء مع أرقام واتساب لاختيارهم عند إرسال التقارير.

### ربط Easy Order
- إدخال **مفتاح API** من لوحة Easy Order في الإعدادات.
- جلب تفاصيل أي طلب برقم الطلب (Order ID) وإضافته كإيراد في الحسابات تلقائياً.

### إرسال التقارير عبر واتساب
- **من صفحة التقارير**: إرسال «ملخص الرصيد» أو «تقرير شهري» لرقم العميل — يفتح واتساب مع النص جاهزاً للإرسال.
- **من الفواتير**: زر «واتساب» بجانب كل فاتورة لإرسال تفاصيلها للعميل على رقمه.

### استيراد من Excel
- **استيراد حسابات قديمة**: من صفحة «استيراد Excel» ارفع ملف Excel (.xlsx, .xls) أو CSV من حساباتك القديمة.
- التعرف التلقائي على الأعمدة (التاريخ، النوع، المبلغ، الوصف، الفئة) أو تعيينها يدوياً.
- معاينة الحركات قبل الإضافة ثم إضافتها كلها دفعة واحدة للتطبيق.
- قالب CSV نموذجي متوفر للتحميل من نفس الصفحة.

### لأصحاب المحلات والتجارة الإلكترونية
- **بيع سريع**: من لوحة التحكم سجّل مبيعة بضغطة (مبلغ + وصف اختياري).
- **كشف يومي**: رصيد افتتاح، إيرادات ومصروفات اليوم، الرصيد المتوقع في نهاية اليوم.
- **هدف مبيعات الشهر**: حدده من الإعدادات واعرض نسبة التحقق على لوحة التحكم.
- **رسم بياني**: إيرادات ومصروفات آخر 6 أشهر.
- **تنبيه فواتير مستحقة**: إظهار الفواتير المستحقة خلال 3 أيام.
- **فئات مناسبة**: شراء بضاعة، شحن، إيجار، رواتب، تسويق، إلخ.
- **نسخ احتياطي**: تصدير كل البيانات (JSON) أو Excel، واستعادة من نسخة سابقة.

### المساعد الذكي (ذكاء اصطناعي)
- **رؤى ذكية**: على لوحة التحكم تظهر تلقائياً نصائح مبنية على بياناتك (مثل: المصروفات تتجاوز الإيرادات، نسبة هدف المبيعات، فواتير مستحقة، تحسين المبيعات).
- **المساعد الذكي**: صفحة مخصصة لتسأل بالعربية عن حساباتك (مثال: كم مبيعات الشهر؟ نصائح لتحسين الرصيد). يعمل مع **OpenAI** (ChatGPT) أو **Google Gemini** بعد إضافة مفتاح API من الإعدادات.
- الإعدادات: اختر المزود (OpenAI / Gemini) وأدخل المفتاح ثم احفظ.

### لوحة الأدمن والتخصيص
- من القائمة اختر **أدمن** وأدخل كلمة المرور (الافتراضية: `admin123`، يمكن تغييرها من داخل لوحة الأدمن).
- **تعديل التطبيق**: تغيير اسم التطبيق والعنوان الفرعي، رفع **لوجو**، رفع **صورة بانر** تظهر في لوحة التحكم. الصور تُحفظ محلياً.

### الاشتراكات وربط واتساب
- صفحة **الاشتراكات**: خطة شهرية (99 ج.م) وخطة سنوية (999 ج.م مع توفير). زر «اشترك عبر واتساب» يفتح محادثة واتساب مع رسالة جاهزة.
- من **الإعدادات** أدخل **رقم واتساب للتواصل** (للاستفسارات والاشتراك). يظهر في صفحة الاشتراكات وزر «تواصل معنا على واتساب».

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح المتصفح على العنوان الذي يظهر (عادةً `http://localhost:5173`).

## البناء للإنتاج

```bash
npm run build
```

البيانات تُحفظ محلياً في المتصفح (localStorage). يمكن نقل التخزين إلى **Supabase** (المرحلة 1): اتبع **خطوة بخطوة** الملف **`docs/خطوات_ربط_Supabase.md`** (إنشاء المشروع، نسخ الرابط والمفتاح، إنشاء `.env`، تشغيل الـ migration). للمراحل التالية (Auth، Multi-tenancy، الدفع) راجع `docs/MIGRATION_PHASES.md`.

## الحصول على مفتاح Easy Order

1. ادخل إلى لوحة Easy Order (Seller Dashboard).
2. من **الإعدادات** → **Public API** → **Create New API Key**.
3. انسخ المفتاح والصقه في إعدادات التطبيق (صفحة الإعدادات).

# السوق المستهدف — محاسب مشروعي

## من هم المستهدفون؟

### 1) الفئة الأساسية (الأولوية الأولى)

| الوصف | أمثلة | الحجم التقريبي في مصر |
|-------|-------|-------------------------|
| **محلات تجزئة صغيرة** | بقالة، ملابس، إلكترونيات، عطارة | ملايين المنشآت الصغيرة جداً |
| **مشاريع منزلية / حرفيون** | خياطة، صيانة، تركيب، طبخ وبيع | مئات الآلاف |
| **بائعون أونلاين (سوشيال + واتساب)** | بيع على فيسبوك/انستغرام + واتساب للطلبات | مئات الآلاف |
| **مستقلون ومقدمون خدمات** | استشاريون، مصممون، محترفون يصدّرون فواتير | عشرات الآلاف |

**الخصائص المشتركة:**
- إيرادات شهرية تقريباً من بضعة آلاف إلى 100 ألف ج.م.
- لا يستخدمون برامج محاسبة رسمية (زوهو، كويك بوكس، إلخ) إما لسعرها أو لتعقيدها.
- يعتمدون على دفاتر أو Excel أو واتساب فقط.
- يريدون: معرفة الرصيد، من يدينني ومن أديان، فواتير بسيطة، تقارير للشريك أو للضرائب لاحقاً.

**المنطقة الجغرافية:** مصر (السوق الأول)، ثم أي سوق عربي يدفع بالعملة المحلية ويهتم بالمحاسبة البسيطة.

## حجم السوق (تقدير لأجل الدراسة)

- **عدد المنشآت الصغيرة جداً والصغيرة في مصر:** مئات الآلاف إلى أكثر من مليون (حسب تعريف الجهاز المركزي وغيره).
- **من يستخدم تطبيق محاسبة على الموبايل:** نسبة صغيرة حالياً، لكن النمو في الاعتماد على الموبايل والدفع الإلكتروني يزيد الفرصة.
- **السوق القابل للاستهداف في السنة الأولى (واقعي):** آلاف المستخدمين المسجلين، ومئات المدفوعين إذا كان التسويق والاحتفاظ جيدين.

## احتياجات السوق التي يلبيها التطبيق

| الاحتياج | كيف يلبيها التطبيق |
|----------|---------------------|
| معرفة الرصيد الحالي | لوحة تحكم + تقرير يومي |
| تنظيم الإيرادات والمصروفات | حركات + فئات جاهزة |
| فواتير للعميل | إصدار فاتورة + إرسال واتساب |
| تذكير بالمديونيات | تنبيه فواتير مستحقة + صفحة ديون |
| ربط الطلبات أونلاين | Easy Order، Shopify، API مواقع |
| تقارير شهرية أو للشريك | تقارير + إرسال واتساب |
| دفع محلي بسيط | فودافون كاش وانستا باي |
| واجهة عربية بسيطة | RTL، مصطلحات محاسبة مألوفة |

## من ليس مستهدفاً (في المرحلة الحالية)

- الشركات المتوسطة والكبيرة (لديها محاسب أو برامج ERP).
- من يبحث عن محاسبة ضريبية كاملة متوافقة 100% مع القانون دون تدخل محاسب.
- من يريد فقط تطبيق دفع أو نقطة بيع (POS) بدون تركيز على الحسابات والتقارير.

## الخلاصة

**السوق المستهدف بإيجاز:** أصحاب محلات ومشاريع صغيرة وبائعون أونلاين ومستقلون في مصر (والعالم العربي)، إيراداتهم محدودة، ويريدون تنظيم الحسابات والفواتير والتقارير من الموبايل بدفع محلي (فودافون كاش / انستا باي) وسعر شهري منخفض.

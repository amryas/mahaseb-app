# إعلان بيانات التطبيق (Data safety) — Google Play

في **Play Console** → تطبيقك → **سياسة التطبيق** → **البيانات التي يجمعها التطبيق**، استخدم الإجابات التالية.

---

## هل يجمع التطبيق أو يشارك أيًا من أنواع البيانات المطلوبة؟

**نعم.**

---

## أنواع البيانات (اختر التالي)

| النوع في القائمة | اختر | ملاحظة |
|------------------|------|--------|
| **المعلومات الشخصية** | ✅ | |
| — البريد الإلكتروني | ✅ | لتسجيل الدخول |
| — معلومات الحساب | ✅ | اسم/بيانات الحساب الذي ينشئه المستخدم (مشاريع، أسماء شركات) |
| **البيانات المالية** | ✅ | |
| — معلومات المعاملات المالية | ✅ | إيرادات، مصروفات، فواتير، مبيعات (يدخلها المستخدم) |
| **معالجة الدفع** | ✅ | عند الاشتراك عبر فودافون كاش / انستا باي (معالجة عبر Paymob) |

---

## هل يتم جمع أو مشاركة البيانات لأغراض التتبع؟

**لا** — إلا إذا استخدمت لاحقاً إعلانات أو تحليلات تتبع عبر جهات خارجية؛ حينها حدّث الإعلان.

---

## لماذا يتم جمع البيانات؟

- **تقديم الخدمة:** تشغيل التطبيق، المزامنة، الدفع.
- **إدارة الحساب:** تسجيل الدخول والمصادقة.

---

## هل يتم مشاركة البيانات مع أطراف ثالثة؟

- **نعم، مع مقدّم الخدمة (معالجة البيانات):** تخزين البيانات على Supabase ومعالجة الدفع عبر Paymob.
- **لا** لمشاركة البيانات لأغراض الإعلان أو بيع البيانات.

---

## هل يمكن للمستخدم طلب حذف البيانات؟

**نعم** — اذكر في الوصف أن المستخدم يمكنه التواصل لطلب حذف حسابه أو تصدير بياناته (وفق سياسة الخصوصية).

---

## أمان البيانات

- **البيانات مُشفّرة أثناء النقل:** نعم (HTTPS).
- **يمكن للمستخدم طلب حذف البيانات:** نعم.

املأ الاستبيان في Play Console وفق هذه النقاط. إن ظهرت لك عبارات مختلفة قليلاً، اختر الأقرب لهذا الوصف.
